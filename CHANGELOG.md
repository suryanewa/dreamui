# DreamUI Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial monorepo setup with pnpm workspaces
- Foundational design tokens (colors, typography, spacing, shadows, motion)
- Theme system with light/dark mode support
- Multiple color palette variants (rose, orange, amber, emerald, sky, violet, fuchsia, pink)
- Utility classes (visibility, focus rings, containers, grids, glassmorphism)
- UI component base templates (button, badge, card, modal, tabs, etc.)
- Form component base templates (input, select, checkbox, switch, slider, etc.)
- Layout component templates (container, stack, grid, surface, glass-panel)
- Media component templates (aspect-ratio, image, video-embed)
- Effect component templates (glow, shimmer, blur-orb)
- Next.js documentation app structure
- GitHub Actions CI/CD workflow
- Placeholder React wrapper package

### Changed
- Updated tooling baseline (Prettier, ESLint, Stylelint)
- Configured TypeScript for strict mode

### Fixed
